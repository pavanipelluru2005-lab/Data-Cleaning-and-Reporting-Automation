# Data Cleaning and Reporting Automation Using Excel

## Objective
Automate data cleaning and reporting workflows using Excel/Google Sheets.

## Tools Used
- Google Sheets
- Pivot Tables
- Charts

## Tasks Performed
- Identified missing values
- Filled missing values
- Standardized inconsistent city names
- Removed duplicate records
- Created Pivot Table reports
- Generated Column and Pie Charts

## Results
The dataset was successfully cleaned and analyzed. Automated reporting improved data quality and reporting efficiency.
